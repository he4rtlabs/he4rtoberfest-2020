**Como rodar o desafio**:

Use o comando abaixo:

```bash
node media.js arg1, arg2 arg3...
```

**Exemplo**:

```bash
node media.js 10 5 7
```

O exemplo irá retornar: Média => 7.33

Lembrando que os valores são as notas que deseja calcular a média
